42 /** Basic function to detect multiple objects (or the same object at different scales) on the same input image using HOG. This function calls the following NEON kernels:
43  *
44  * -# @ref NEHOGGradient
45  * -# @ref NEHOGOrientationBinningKernel
46  * -# @ref NEHOGBlockNormalizationKernel
47  * -# @ref NEHOGDetector
48  * -# @ref CPPDetectionWindowNonMaximaSuppressionKernel (executed if non_maxima_suppression == true)
49  *
50  * @note This implementation works if all the HOG data-objects within the IMultiHOG container have the same:
51  * -# Phase type
52  -# Normalization type
53  -# L2 hysteresis threshold if the normalization type is L2HYS_NORM
54  *
55  */

65  /** Initialise the function's source, destination, detection window strides, border mode, threshold and non-maxima suppression
66  *
67  * @param[in, out] input Input tensor. Data type supported: U8
68  * (Written to only for @p border_mode != UNDEFINED)
69  * @param[in] multi_hog Container of multiple HOG data object. Each HOG data object describes one HOG model to detect.
70  * This container should store the HOG data-objects in descending or ascending cell_size width order.
71  * This will help to understand if the HOG descriptor computation can be skipped for some HOG data-objects
72  * @param[out] detection_windows Array of @ref DetectionWindow used for locating the detected objects
73  * @param[in] detection_window_strides Array of @ref Size2D used to specify the distance in pixels between 2 consecutive detection windows in x and y directions for each HOG data-object
74  * The dimension of this array must be the same of multi_hog->num_models()
75  * The i-th detection_window_stride of this array must be multiple of the block_stride stored in the i-th multi_hog array
76  * @param[in] border_mode Border mode to use.
77  * @param[in] constant_border_value (Optional) Constant value to use for borders if border_mode is set to CONSTANT.
78  * @param[in] threshold (Optional) Threshold for the distance between features and SVM classifying plane
79  * @param[in] non_maxima_suppression (Optional) Flag to specify whether the non-maxima suppression is required or not.

83  */
