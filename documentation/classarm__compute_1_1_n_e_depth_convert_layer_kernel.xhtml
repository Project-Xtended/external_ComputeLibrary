Detailed Description

Depth conversion kernel This function ignores the scale and zeroPoint of quanized tensors, i.e.

QASYMM8 input is treated as uint8 values.

Parameters
[in]inputThe input tensor to convert. Data types supported: QASYMM8_SIGNED/QASYMM8/U8/U16/S16/F16/F32.
[out]outputThe output tensor. Data types supported: QASYMM8_SIGNED/QASYMM8/U8/U16/S16/U32/S32/F16/F32.
[in]policyConversion policy.
[in]shift(Optional) Value for down/up conversions. Must be 0 <= shift < 8.
